Методы

new enigne("canvas")


пример:var x = new enigne("canvas")

аргумент(ы):1 arg) пишите название id вашего канваса

Описание:создает поле для взаимодействия с движком

phys(gravity,speed)


пример:x.phys(0.5,15)

аргумент(ы):1 arg)сила гравитации

аргумент(ы):2 arg)скорость обработки гравитации

Warning:лучше не меняйте второй аргумент если ещё не хорошо понимаете механику движка

Описание:даёт блокам физические или статические свойства гравитации

sysphys(gravity,speed)


пример:x.sysphys(0.5,15)

Аналог:phys(gravity,speed)

разница:именно этот метод вызывается из самого ядра а не из физического движка

аргумент(ы):1 arg)сила гравитации

аргумент(ы):2 arg)скорость обработки гравитации

Warning:лучше не меняйте второй аргумент если ещё не хорошо понимаете механику движка

Описание:даёт блокам физические или статические свойства гравитации

x.block('id','type',x,y,w,h,'color');


пример:x.block('player','player',150,100,25,25,'purple');

аргументы

  1. id:через этот идинтификатор будете к нему подключаться
  2. type:player|block|button
  3. x:указать координату x
  4. y:указать координату y
  5. w:указать ширину
  6. h:указать высоту
  7. 'color':=цвет можно задавать:black|#123321|rgb('50','50','50')

Warning:первый блок должен быть указан типом игрока это обязательно а потом уже другие блоки любого типа

Warning 2:x.block('but1','button',1050,25,100,25,'black','green','текст',function (){
alert("ds");
})
задаётся с допольнительными аргументами текста и телом функции

Описание:создаем окружение в виде блоков

clear()


пример:x.clear()

Warning:внимание может не полностью стирать если canvas слишком большой

Описание:принудительно стирает все блоки

render()


пример:x.render();

Описание:принудительно рендерит все блоки которые были обьявлены

v.x('id')


пример:let px = x.v.x('player');||console.log(x.v.x('block'));

аргументы

  1. id:название блока(вводим id)

показывает на какой координате x находится блок

v.y('id')


пример:let px = x.v.x('player');||console.log(x.v.x('block'));

аргументы

  1. id:название блока(вводим id)

показывает на какой координате y находится блок

move(id,x,y)


пример:x.move("player",15,15);

аргументы

  1. id:название блока(вводим id)
  2. x:вводим координату x для перемещения
  3. y:вводим координату y для перемещения

телепортирут(а может плавно переместить) блок или игрока(смотря что указано в id)
по координатам указанным в x,y

new VECTOR2(x,y)


пример:var test = new VECTOR2(3,5);

аргументы

  1. x:вводим координату x для его последующего использования
  2. y:вводим координату y для его последующего использования

удобное хранение x,y координат в виде вектора

События

block('id','button',x,y,w,h,color,text,fun())


x.block('but1','button',1050,25,100,25,'black','green','текст',function (){
alert("ds");
})

аргументы

  1. id:создаем идентификатор для будущего подключения
  2. button:указываем обязательно тип button
  3. x:указываем координату x
  4. y:указываем координату y
  5. w:ширина кнопки
  6. h:высота кнопки
  7. color:цвет заднего фона кнопки
  8. text:текст внутри кнопки
  9. fun():функция которая будет обрабатыватся при нажатии на кнопку

если правильно заполнить то при нажатии на кнопку сработает событие описанное в fun()

события и управление

key(key_en,key_ru,fun())


пример:key('a','ф',(()=>{
py = x.v.y('player')
px-=10;x.move("player",px,py);
}))

аргументы

  1. key_en:клавиша в английской раскладке если будет нажата сработает событие
  2. key_ru:клавиша в русской раскладке если будет нажата сработает событие
  3. fun():вводим координату y для перемещения

если правильно заполнить то при нажатий клавиш будет срабатывать событие
